SECTION 1
The external libraries used are WTForms and Flask.
Flask is used to contain all of the code needed to run a python3 web server.
WTForms is just used to prepare and standardise the forms into manageable code.

SECTION 2
To run the server just use, "python3 run.py" (ignoring quotation marks)
The URL of the homepage should be localhost:5000

SECTION 3
To run test cases, run the py files.
